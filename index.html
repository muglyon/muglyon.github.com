Microsoft User Group Lyon

L'association

Le Microsoft User Group de Lyon, c'est un ensemble de professionnels de l'informatique qui travaillent autour des technologies Microsoft. Ils sont issus de tous les horizons (formateurs, développeurs, chefs de projet, architectes, etc).

Le collectif

Le MUG Lyon est une association collegiale qui comporte un noyau dur d'organisateurs et de passionnés qui se ferons un plaisir de répondre à vos questions et de vous guider dans la communauté.

Atelier "Software Factory sur Azure" + Afterwork - organisé en novembre 2013

Le MUG Lyon organise un atelier "Software Factory avec Azure" hébergé par Sciences-U, suivi d'un afterwork; ces évènements sont gratuits et ouverts à tou(te)s.

Industrialisez vos développements avec la puissance du Cloud Azure et de l’"ALM Microsoft version 2013" (1).
Véritables retours d’expériences basés sur un usage quotidien sur de multiples projets.
Découvrez comment 45 min suffisent pour créer un projet Web, mettre en œuvre sur le Cloud Azure une véritable "Software Factory" (2) de qualité et déployer l'ensemble, toujours sur Windows Azure.

(1) "ALM Microsoft version 2013": gestion du cycle de vie des applications, basé sur Visual Studio 2013 et TFS 2013.

Atelier "Qualité de code avec Sonar" + Afterwork - organisé en octobre 2013

Le MUG Lyon organise un atelier "Qualité de code avec Sonar" hébergé par Sciences-U, suivi d'un afterwork; ces évènements sont gratuits et ouverts à tou(te)s.

Qui ne s'est jamais plaint d'un code existant ? Mais pour autant sur quelles mesures vous basez-vous pour ce jugement ? Cet atelier vous propose de revisiter les fondements de la qualité de code, c'est-à-dire des métriques et des normes de code. Cela sera illustré au travers de l'utilisation de l'outil Sonar (aka SonarQube) permettant le suivi de la qualité (de code C#, VB.NET biensûr, mais de plein d'autres langages). L'atelier finira par quelques conseils d'utilisation dans des contextes courants.

Atelier "ActionTypes" + Afterwork - organisé en septembre 2013

Le MUG Lyon organise un atelier "ActionTypes" hébergé par Sciences-U, suivi d'un afterwork; ces évènements sont gratuits et ouverts à tou(te)s.

Ne vous êtes-vous jamais demandé comment améliorer votre communication et votre relationnel ?
Ne vous êtes-vous jamais demandé comment « fonctionne » votre interlocuteur ?

Dominique LOPEZ, Consultant Formateur et Coach en développement personnel, vous propose un atelier basé sur l’approche Action|Types®.

Venez découvrir ce formidable outil de connaissance de soi basé sur la découverte des correspondances entre la motricité, la vision et les préférences d'utilisation des zones du cerveau.
Cette méthode permet de comprendre et d’accepter les différences fondamentales entre les êtres humains, de reconnaître comment un interlocuteur préfère être approché et sur quelles bases il acquiert l’information et prend ses décisions.
Intégrer ces différences constitue les premières étapes de tout processus d’interaction, de communication et de performance.

Atelier "Rebase, You Git !" + Afterwork - organisé en juin 2013

Le MUG Lyon organise un atelier "Rebase, You Git !" hébergé par Sciences-U, suivi d'un afterwork; ces évènements sont gratuits et ouverts à tou(te)s.

Cette présentation vous donnera les clés pour mieux comprendre Git, un outil de gestion de sources décentralisé, créé en 2005 par Linus TORVALDS en 2005.

Nous aborderons les concepts de base de Git, pour mettre rapidement les mains dans le cambouis et utiliser Git de manière efficace. Nous étudierons les bonnes pratiques et le concept de BYOW (Bring Your Own Workflow) pour définir un workflow durable. Nous terminerons par une présentation de GitHub, une plateforme de social coding basée autour de Git pour structurer notre workflow.

Enfin, cet atelier sera suivi par un afterwork, au cours duquel nous pourrons continuer d'échanger autour d'un verre (attention, toutes les consommations sont à la charge des participant(e)s).
